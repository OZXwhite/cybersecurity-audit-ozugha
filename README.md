# Beginner Cybersecurity Audit Project

## Name
David Ozugha

## Project Title
Personal Laptop Cybersecurity Audit

## Objective
The aim of this project is to identify basic cybersecurity weaknesses in a personal device and suggest practical improvements.

## Scope
I audited my personal laptop.

## Tools Used
- Windows Settings
- Windows Security
- Control Panel
- Browser settings

## Files in this Repository
- checklist.md
- findings-report.md
- recommendations.md
- screenshots/

## Key Findings
- Password protection was enabled
- Antivirus was active
- Firewall was enabled
- The operating system was not fully updated
- All installed applications were recognized and in use
- Browser passwords were well protected

## Conclusion
This project helped me understand how to identify cybersecurity risks and improve device security.
